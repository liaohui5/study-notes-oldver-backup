## Nginx 是什么

Nginx 是一个高性能的 web 服务器软件, 类似的还有 [candy](https://caddyserver.com/) [Apache](https://httpd.apache.org/) 等

## 能做什么? 为什么要学习他?

1. 可以部署前端打包后的今天资源, 提供访问
2. 学习其他的两个也是可以的

## 安装

```bash
# 可以用包管理工具一键安装
yum install -y nginx

# or 
apt-get install nginx
```
